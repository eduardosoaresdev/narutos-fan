SOBRE O PROJETO

Este site é como uma rede social para fans do anime Naruto, que carinhosamente dei o nome de Naruto's Fan.
O intuito é colocar o que aprendi de front-end na prática, utilizando dos principais conceitos de CSS que vi no curso até o momento:

  1. CSS Grid
  2. CSS Flexbox
  3. SASS
    • Nesting;
    • Mixin;
    • Minify;
  4. Design responsivo

Neste projeto, além dessa Home page, você verá mais 3 telas, são elas:
